# CS333 - Network Security and Defense In-Class Exercises

Please fork this repository and clone it to your local machine. This repository will contain all the in-class exercises for CS333 - Computer Networks. As we cover different topics, I'll be adding scripts and code snippets here for you to work with.

You will be expected to complete the exercises during class, and you can also use this repository to practice and experiment with the code outside of class. Make sure to commit your changes regularly and push them to your forked repository.

You will be graded on your participation and the completion of these exercises, so make sure to stay engaged and ask questions if you need help. You will submit your work by creating a pull request to the main repository.

## Exercise 1: Ping Sweep Script

In this exercise, you will create a simple ping sweep script using Bash. The script will take a range of IP addresses and check which ones are active by sending ICMP echo requests (ping).
